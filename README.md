Overview
Goal: Help students discover, filter, and save college events in one place.

  This is a code bundle for Campus Event Finder App. The original project is available at https://www.figma.com/design/Wd69leFsxW9MCxla5CueAC/Campus-Event-Finder-App.

Design Specs:

Desktop layout (1440px width)
Grayscale only
Rectangles and text only
Focus on layout and usability

Screens

Landing / Events Page - Navigation bar, search bar, filter toggle, and event cards grid
Event Details Page - Event image, title, date, venue, description, register/save buttons
Filter Section - Category checkboxes, date/location filters, apply button
My Events Page - List of saved events with details and action buttons
  ## Running the code

  Run `npm i` to install the dependencies.
